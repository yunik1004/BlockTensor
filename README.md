> 2018 Fall Semestor CS408 Project

## Build

### Dependencies
- NPM

### Debug
You need at least two shell to run the application.

For the frontend server,
```bash
$ cd frontend
$ npm run dev
```

For the backend server,
```bash
$ cd backend
$ DEBUG=backend*
$ npm start
```

### Release
You need only one shell to run the application.

```bash
$ cd frontend
$ npm run build
$ cd ../backend
$ npm start
```
